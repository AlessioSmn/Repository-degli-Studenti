	FUNZIONALITA'

Pagina web per l'upload e la consultazione di documenti relativi all'università, idealmente appunti degli studenti resi pubblici dagli stessi.
	OPTIONAL | Pagina per la valutazione MA servirebbe un controllo sulle valutazioni (es: evitare spam di cattive valutazioni)


	SITO

sito usufruibile solo dopo il login, quindi INDEX(initial)=>LOGIN

INDEX: => pagina di login/signup

LOGIN: email-password => HOME(vera) (/ recupera password)
SIGNUP: registrazione completa (nome, cognome, email, password, ...) => HOME(vera)

SEARCH: ricerca di appunti secondo dipartimento o materia, scegli l'ordinamento (facile con query, )
PERSONAL: area riservata per la modifica/cancellazione/inserimento di documenti (pdf, altro?)
	
	
	FEATURES

Modalita giorno/notte gestita via JS (https://techinscribed.com/multiple-themes-using-css-variables/)
Preferenza della modalità salvata SE possibile (magari con cookie)
Modalità scelta MANTENUTA quando si cambia pagina, almeno finchè non si chuide


	DATABASE

user(id, name, surname, email, password)

{ Universita(id, Nome) } | OPTIONAL se faccio non solo unipi => in quel caso togli la mail istituzionale e mettine una generica

degreecourse(id, name)
subject(id, name, cfu, year, degreecourse)

Documento(id, title, subtitle|null, uploadDate, lastModified, subject[FK], owner[FK], downloadcounter)
      pdf memorizzato in progetto/docs/[id].[estensione]

OPTIONAL | Valutazione({id}, Data, Utente, Documento, [RatingsVari])
